# 设计和制作 浙江省粮食局门户网站

## 要求

- 一个主页/七个二级页面/七个三级页面
- 至少两个动态设计

## 频道

- 首页（“粮油在线LOGO”、”浙江省粮食局政务网”文字、省政府网站统一标志）
  - 图片新闻（图片、文字报道）
  - 网上办事（行政许可项目、办事指南、表格下载、状态查询）
  - 互动交流（局长信箱、在线咨询、投诉举报、下载中心）
  - 用户登录（用户注册、登录区域）
  - 信息查询（以一个或多个关键字的方式查询网站内容）
  - 公示公告（竞价交易、人事任免、部门预决算）
  - 网站链接（省政府及省级部门网站的链接、市县粮食局网站、粮食行业网站等的链接）
